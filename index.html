Lightweight Speech Enhancement Model Based on Harmonic Attention and Phase Estimation with Skin-Attachable Accelerometer

Introduction

Welcome to the official page of a lightweight ACC-assisted U-Net (LAU-Net). This model enhances speech in noisy environments using harmonic attention and phase estimation techniques. The code for this project can be found at https://github.com/LAU-Net/LAU-Net.git.

  2. L. Yang, W. Liu, R. Meng, G. Lee, S. Baek, and H.-G. Moon, “Fspen: an ultra-lightweight network for real-time speech enhancement,” in IEEE International Conference on Acoustics, Speech and Signal Processing (ICASSP), 2024, pp. 10 671–10 675.
